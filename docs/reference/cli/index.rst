Command Line
============

Start
:::::

Starts local dev services using honcho (only for development purposes).

.. code-block:: bash

    $ harp start [--with-docs/--no-docs] [--with-ui/--no-ui] [service1, service2 ...]

Available services:

- frontend
- proxy
- docs
- ui
